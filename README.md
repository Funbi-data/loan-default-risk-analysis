# Credit Risk Assessment & Loan Default Prediction

This project develops a machine learning-based credit risk model to predict loan default and support financial decision-making.

## Project Overview
The model identifies high-risk borrowers using financial and credit-related features. It addresses class imbalance using SMOTE and improves detection of defaulters.

##  Key Features
- Loan default prediction using Machine Learning
- Improved recall using SMOTE
- Feature importance analysis
- Batch risk scoring system for multiple applicants
- Business decision recommendations (Approve, Review, Reject)

##  Key Insights
- Credit policy, interest rate, and FICO score are strong predictors of default
- SMOTE significantly improved detection of high-risk borrowers
- The model performs best as a decision-support tool

##  Business Value
This project simulates a real-world credit risk scoring system that:
- Evaluates multiple applicants at once
- Assigns default probabilities
- Segments applicants into risk categories
- Supports faster and more consistent loan decisions
